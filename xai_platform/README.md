# XAI Model Explanation Platform

## Project Purpose
A platform that takes a trained sklearn-compatible ML model and a tabular dataset to generate explainability reports using SHAP.

## Setup Instructions
1. Clone the repository.
2. Create and activate a virtual environment (optional but recommended).
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## How to Run the App
To start the Streamlit application, run the following command from the `xai_platform` directory:
```bash
streamlit run frontend/app.py
```
